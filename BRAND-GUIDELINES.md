# NEGA 品牌指南 (Brand Guidelines)

## 核心品牌信息

### 产品名称
**NEGA** - 您的AI英文语法助手和西海岸纯正口语教练

### 品牌标语 (Slogans)
**英文**: Your Native English Grammar Assistant  
**中文**: 你的西海岸纯正口语教练

### 品牌使命
帮助全球英文学习者通过AI技术实现自然流畅的英文表达，提供专业级的语法纠正和口语指导。

### 品牌价值观
- 🎯 **准确性**: 精准的语法检查和纠正
- 🌍 **包容性**: 支持多语言学习（英文-中文）
- 🚀 **创新性**: 采用最新的AI和NLP技术
- 💪 **赋能性**: 帮助学习者自信说英文

- ✨ **友好的**: 亲近、有帮助感

## 使用规范 (Guidelines for Usage)

### Logo使用规范

#### ✅ 正确使用
```
- 使用标准配色
- 保持最小尺寸 (最小宽度 100px)
- 清晰的背景对比
- 完整的标志 (不拆分)
- SVG或高分辨率PNG格式
```

#### ❌ 禁止使用
```
- 改变颜色或配色
- 变形或拉伸logo
- 旋转或倾斜logo
- 添加效果或阴影
- 使用低分辨率版本
- 放在复杂背景上（降低对比）
```

### 清晰空间 (Clear Space)
```
Logo周围需要保留清晰空间，至少为logo高度的 25%
```

### 背景使用
```
推荐背景:
✓ 纯白色
✓ 品牌蓝 (#0066FF)
✓ 浅灰色背景

避免:
✗ 彩色渐变
✗ 照片背景
✗ 图案背景
```
