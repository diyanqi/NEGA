Riva gRPC protos

Place the NVIDIA Riva proto files in this folder:
- riva_asr.proto
- riva_audio.proto
- riva_common.proto

These files are required by the gRPC client in:
- src/app/api/transcribe/route.ts

You can download them from the official NVIDIA Riva "common" repository
under riva/proto. Keep the same filenames.
